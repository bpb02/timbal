# Installation

<h2 style={{ fontSize: '1.2rem', fontWeight: 'normal', color: 'var(--ifm-font-color-base)' }}>
  View the latest updates and changes to Timbal
</h2>

:::info[Python Version Requirements]{.tight-admonition}
**Timbal** requires `Python >= 3.12`.

Make sure you have the correct version installed before proceeding.
:::

Timbal uses the `uv` for dependency management and package handling, streamlining project setup and execution for a seamless development experience.

:::note[Recommendation]
Install uv to manage your Python dependencies.

- On macOS/Linux:
Use `curl` to download the script and execute it with `sh`:

<CodeBlock language="bash" code ={`curl -LsSf https://astral.sh/uv/install.sh | sh`}/>

If your system doesn't have `curl`, you can use `wget`:
<CodeBlock language="bash" code ={`wget -qO- https://astral.sh/uv/install.sh | sh`}/>
- On Windows:
Use `irm` to download the script and `iex` to execute it:
bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

If you run into any issues, refer to **[UV's installation](https://docs.astral.sh/uv/get-started/installation/)** guide for more information.
:::


### Install Timbal

Run the following command to install Timbal CLI:

<CodeBlock language="bash" code ={`uv add timbal`}/>

:::success[Success]
You're ready to create your first Timbal project!
:::
